# SirRezi Banking System

A modern, clean, and optimized banking system for FiveM using ESX Legacy, ox_target, and ox_lib.

## Features
- **Modern UI**: Clean, responsive, and user-friendly interface with dark mode.
- **Optimized**: Uses `ox_target` for interaction and `ox_lib` for notifications and utilities.
- **ATM Support**: Automatically detects all ATM models in the game world.
- **Bank Peds**: Configurable bank locations with NPC tellers.
- **Secure**: Server-side validation for all transactions.

## Installation

1. **Dependencies**: Ensure you have the following resources installed and started:
   - `es_extended` (Legacy)
   - `ox_lib`
   - `ox_target`

2. **Download**: Place the `SirRezi_Banking` folder into your server's `resources` directory.

3. **Configuration**: 
   - Open `config.lua` to add more bank locations or change settings.
   - Default bank location is set to Fleeca Bank at `vec3(149.4713, -1042.1696, 29.368)`.

4. **Server Config**: Add `ensure SirRezi_Banking` to your `server.cfg`.

## Usage
- **Banks**: Walk up to the bank teller NPC and use `Alt` (or your target key) to open the bank menu.
- **ATMs**: Walk up to any ATM and use your target key to access the ATM menu.

# SirRezi Banking System (Deutsch)

Ein modernes, sauberes und optimiertes Banksystem für FiveM basierend auf ESX Legacy, ox_target und ox_lib.

## Funktionen
- **Modernes UI**: Saubere, responsive und benutzerfreundliche Oberfläche im Dark Mode.
- **Optimiert**: Nutzt `ox_target` für Interaktionen und `ox_lib` für Benachrichtigungen.
- **ATM Support**: Erkennt automatisch alle ATM-Modelle in der Spielwelt.
- **Bank Peds**: Konfigurierbare Bankstandorte mit NPC-Angestellten.
- **Sicher**: Serverseitige Validierung aller Transaktionen.

## Installation

1. **Abhängigkeiten**: Stelle sicher, dass folgende Ressourcen installiert und gestartet sind:
   - `es_extended` (Legacy)
   - `ox_lib`
   - `ox_target`

2. **Download**: Platziere den `SirRezi_Banking` Ordner in dein `resources` Verzeichnis.

3. **Konfiguration**: 
   - Öffne `config.lua`, um weitere Bankstandorte hinzuzufügen oder Einstellungen zu ändern.
   - Standardmäßig ist die Fleeca Bank bei `vec3(149.4713, -1042.1696, 29.368)` konfiguriert.

4. **Server Config**: Füge `ensure SirRezi_Banking` zu deiner `server.cfg` hinzu.

## Nutzung
- **Banken**: Gehe zum Bankangestellten NPC und nutze `Alt` (oder deine Target-Taste), um das Bankmenü zu öffnen.
- **ATMs**: Gehe zu einem beliebigen ATM und nutze deine Target-Taste, um das ATM-Menü zu öffnen.
